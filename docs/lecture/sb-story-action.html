# Story 액션

액션(action)은 Story에서 이벤트 핸들러(callback) 인수가 받은 데이터를 표시하는 데 사용됩니다.

# 액션 인자

액션은 action 인자를 스토리에 제공하여 작동합니다. 인자를 설정하는 방법은 2가지 입니다.

# 사용자 설정

컴포넌트 Story 포멧 (CSF)

export default {
  // ...
  argTypes: { 
    onClick: { action: '클릭!' } 
  }
}

# 액션 자동 감지

액션을 자동 감지하여 추론하도록 하려면 다음과 같이 매개변수를 설정합니다.

export const parameters = {
  actions: { argTypesRegex: '^on[A-Z].*' }
}
