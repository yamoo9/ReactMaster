# Firebase 인증 (회원가입)

Firebase 인증(Authentication) 서비스를 사용해 회원가입 하는 방법을 학습합니다.

# 이메일/패스워드 회원가입

사용자로부터 입력 받은 이메일, 패스워드 정보를 Firebase 인증 서비스에 전달해 새로운 사용자를 생성 할 수 있습니다.



 









auth
  // 입력 받은 이메일, 패스워드 정보를 전달해 인증 사용자 생성 요청
  .createUserWithEmailAndPassword(email, password)
  // 응답(성공) 시, 승인(resolve) 콜백 실행
  .then((credentials) => {
    console.log(credentials.user);
  })
  // 응답(실패) 시, 거절(reject) 콜백 실행
  .catch((error) => {
    console.error(error.message);
  })

회원가입과 동시에 Firestore 데이터베이스 Users 컬렉션에 신규 사용자로 등록하려면 사용자 프로필 생성 유틸리티를 사용합니다. 비동기 함수를 사용해 회원가입 후, 진행되도록 설정할 수 있습니다.








 
 





// 이메일, 패스워드 회원가입 함수 내보내기
export const signUpWithEmailAndPassword = async (
  email,
  password,
  additionalData = {}

  try {
    const { user } = await auth.createUserWithEmailAndPassword(email, password)

# 이메일/패스워드 로그인

가입한 사용자에게 이메일, 패스워드 정보를 입력 받아 로그인 할 수 있도록 설정합니다.


 





// 입력 받은 이메일, 패스워드 정보를 전달해 로그인 요청
auth.signInWithEmailAndPasswornd(email, password)
  // 응답(성공) 시, 승인(resolve) 콜백 실행
  .then({authUser}) => console.log(authUser))
  // 응답(실패) 시, 거절(reject) 콜백 실행
  .catch(error => console.error(error.message))

이메일/패스워드 로그인 폼에서 전달 받을 수 있도록 유틸리티 함수를 정의해 내보낼 수도 있습니다.

// 이메일, 패스워드 로그인 함수 내보내기
export const signInWithEmailAndPassword = async (email, password) => {
  await auth.signInWithEmailAndPassword(email, password)
}
