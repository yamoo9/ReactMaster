# Create React App

create-react-app (opens new window)을 사용하면 Webpack과 같은 빌드 도구를 사용해 프로젝트를 구성하는 데 애먹지 않고, 명령어 1줄로 React 앱 프로젝트를 손쉽게 구성할 수 있습니다. 구성 환경을 갖추는 데 힘 쏟지 않고 앱 개발에만 집중할 수 있습니다.

# React 프로젝트 생성

프로젝트를 생성하는 명령을 입력 실행하여 새로운 프로젝트를 생성할 수 있습니다.

npx create-react-app <프로젝트_이름>
1

# 설치 오류 문제 해결 방법

Creat React App을 사용해 새로운 프로젝트를 생성하였으나, 개발 서버 구동이 안될 경우가 있습니다.

오류 메시지

There might be a problem with the project dependency tree. It is likely not a bug in Create React App, but something you need to fix locally .... If prefer to ignore this check, add SKIP_PREFLIGHT_CHECK=true to an .env file in your project. That will permanently disable this message but you might encounter other issues.

이런 경우 프로젝트 루트 위치에 .env 파일을 만든 후, 아래 코드를 추가하고 저장합니다.

SKIP_PREFLIGHT_CHECK=true
1

오류 원인 및 해결 방법

해당 문제는 의존성 호환 문제로 인해 발생 (opens new window)합니다. 오류가 발생하면 검사를 무시하도록 설정하여 일시적으로 문제가 발생하지 않게 조치한 것입니다. 다만, 검사를 하지 않으므로 다른 문제가 발생할 수 있습니다.

# 명령어 목록

Create React App에서 제공하는 기본 명령은 다음과 같습니다.

명령어 설명
npm start React 프로젝트 개발 서버를 시작합니다.
npm run build 배포를 위해 앱을 정적(Static) 파일로 번들(Bundle) 합니다.
npm run test 테스트 러너(Test Runner)를 시작합니다.
npm run eject 사용자 정의 구성(예: Webpack)을 직접 수행할 수 있도록 프로젝트를 준비합니다.
