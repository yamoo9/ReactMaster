# 문서 헤드 구성

<head> 영역 안에 다음의 콘텐츠를 구성합니다.

  • 환경 변수 .env (opens new window) 설정
  • 퍼블릭(PUBLIC) 구성
  • 검색엔진 최적화(SEO) (opens new window)
  • 오픈 그래프(Open Graph) (opens new window)

# 환경 변수

환경 변수 파일인 env 이름을 .env로 변경합니다.

환경 변수란?

특정 프로세스(process)를 위한 키=값(key=value) 형태의 변수를 말합니다.

Windows 사용자

Windows 운영체제에서는 Sass 경로 설정을 다음과 같이 변경 사용해야 합니다.

SASS_PATH=./node_modules;./src
