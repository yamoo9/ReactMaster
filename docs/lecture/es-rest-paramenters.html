# 전개 연산자

전개 연산자(Spread Operator) (opens new window) ...는 값의 집합(배열) 또는 키:값의 집합(객체)에서 동작하며 집합에 포함된 원소를 전개할 때 사용합니다. 뿐만 아니라, 함수에서는 나머지 매개변수(Rest Paraments) (opens new window)를 표현할 때 사용됩니다.

# ES6

const integers = [-1, 0, 32, -101, 24]

Math.max(...integers)

function plusCount() {
  var first = arguments[0];
  var rest = [].slice.call(arguments, 1);
  
  return rest.reduce(function(sum, next) {
    return sum + next;
  }, first);
}
