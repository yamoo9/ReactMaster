# Firebase 인증(Auth)

Firebase 인증(Authentication) 서비스는 애플리케이션을 이용하는 사용자를 손쉽게 인증하는 다양한 방법을 제공합니다.

# React Firebase 인증

React 또는 React Native 앱에서 Firebase 인증(Authentification)을 쉽게 통합 할 수 있습니다.

# Auth 컴포넌트

React Firebase Auth는 다음의 컴포넌트를 제공합니다.

  • FirebaseAuthProvider
  • FirebaseAuthConsumer
  • IfFirebaseAuthed
  • IfFirebaseAuthedAnd
  • IfFirebaseUnAuthed

# 설치

firebase (opens new window) JS 클라이언트를 설치합니다.

$ npm i firebase
1

@react-firebase/auth (opens new window) 패키지를 설치합니다.

$ npm i @react-firebase/auth
1

# 구성

아래 URL의 프로젝트를 프로젝트 이름으로 변경해 브라우저 주소 창에 입력하면 Firebase 구성 정보를 확인할 수 있습니다.

https://console.firebase.google.com/project/프로젝트/settings/general

Firebase 구성(configuration) 정보 구성은 다음과 같습니다.

const firebaseConfig = {
  apiKey: "API-키-정보",

  authDomain: "인증-도메인-이름",
  databaseURL: "데이터베이스-이름-URL",
  storageBucket: "스토리지-버킷",
  messagingSenderId: "메시징-샌더-아이디",
  appId: "앱-아이디"
};

# FirebaseAuthProvider 컴포넌트

FirebaseAuthProvider 컴포넌트는 firebase props와 firebaseConfig 구성을 전개한 props를 전달 받습니다.

<FirebaseAuthProvider firebase={firebase} {...firebaseConfig}>
  {/* Firebase Auth 정보를 공급 받을 컴포넌트 트리 */}
</FirebaseAuthProvider>

# FirebaseAuthConsumer 컴포넌트

FirebaseAuthConsumer 컴포넌트는 로그인 유무, 사용자, 공급자 ID 등의 정보를 공급 받아 컴포넌트 안에서 사용할 수 있도록 합니다.

<FirebaseAuthConsumer>
  {
    ({isSignIn, user, providerId}) => {
      // 로그인 유무, 사용자, 공급자 ID 확인
      const authInfo = JSON.stringify({isSignIn, user, providerID}, null, 2);
      // 인증 사용자 정보 출력
      return (
        <pre>{authInfo}</pre>
      )
    }
  }
</FirebaseAuthConsumer>

# IfFirebaseAuthedAnd 컴포넌트

IfFirebaseAuthedAnd 컴포넌트는 이름이 의미하는대로 Firebase 인증 사용자인지 검증 후, 결과를 처리합니다. filter props를 통해 특정 조건에 따라 결과 값을 반환할 수 있고 결과 값이 참일 경우에만 화면에 렌더링 합니다.

<IfFirebaseAuthedAnd
  // 필터링
  filter={({ providerId, user }) => {
    // 사용자 이메일 정보가 없다면?
    // false 반환
    if(!user.email){return false;}

    // 사용자 이메일 정보가 있다면?
    // 공급자 ID 값이 'anonymous'가 아니고, 
    // 사용자 이메일에 '@companyname.com'가 포함되어 있다면
    // true 반환, 반대의 경우는 false 반환
    return (
      providerId !== "anonymous" &&
      user.email.includes("@companyname.com")
    );
  }}
>
  {/* FirebaseAuthConsumer 컴포넌트와 동일한 사용법 */}
  {({ isSignedIn, user, providerId }) => {
    return (
      // JSX
    );
  }}
</IfFirebaseAuthedAnd>
