    # Storybook 소개

    Storybook은 React 컴포넌트를 위한 UI 개발 환경입니다. 이를 통해 UI 컴포넌트의 다양한 상태 시각화 또는 인터랙티브 스타일로 개발 할 수 있습니다. Storybook은 앱 외부에서 실행되므로, 앱 모듈 종속성에 대해 걱정하지 않고 UI 컴포넌트를 독립적인 환경에서 개발할 수 있습니다.​​ (참고: Component Driven Development)

    Storybook (opens new window)은 "비즈니스 로직과 컨텍스트"로부터 UI "컴포넌트를 독립적으로 분리"하여 구성할 수 있도록 도와줍니다.

    # 애드온

    ‌Storybook이 기본적으로 제공하는 기능 외에 애드온(Addons) (opens new window)을 추가하여 보다 강력하고 유용하게 활용할 수 있습니다.​
