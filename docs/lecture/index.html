# 강의 소개

야무의 ⎡ React 러닝 가이드 ⎦ 과정은 기본적으로 블렌디드 러닝 방법으로 진행됩니다. 블렌디드 러닝 모집 공고를 통해 과정을 신청할 수 있습니다.

Chrome 웹 브라우저를 사용해 Front-End Master 강의 서비스를 이용하길 권장합니다. (IE, Edge, Firefox 미지원)

온라인 강의만 듣고 싶다면?

블렌디드 러닝 스타일 대신, 온라인 강의만 학습하는 것도 가능합니다.
강의 중 일부는 무료로 볼 수 있지만, 유료 콘텐츠의 경우는 결제 후에 볼 수 있습니다.

# 강의 커리큘럼

좌측 사이드 바의 각 카테고리를 통해 커리큘럼 콘텐츠를 살펴볼 수 있습니다. 참고로 좌측에서 안내된 강의 콘텐츠는 온라인 전용입니다. 블렌디드 러닝 강의는 기본적으로 온라인 강의에 더해 오프라인 강의 콘텐츠를 보다 심도 있게 다루게 됩니다.

온라인 강의 콘테츠는 총 00시간 분량이며, 000개의 동영상을 제공합니다.

스텝 카테고리 설명
00 강의 시작에 앞서 React 프레임워크 학습 내용 및 준비 사항 소개
01 React React 라이브러리 설계 / 개발 방법 학습
02 Styled Component React 컴포넌트 스타일 관리를 위한 Styled Component 학습
03 React Redux React 앱 상태 관리를 위한 Redux 학습
04 React Router 싱글 페이지 애플리케이션(SPA) 제작을 위한 React Router 학습

# 블렌디드 러닝

블랜디드 러닝(Blended Learning)은 2가지 이상의 학습 방법을 결합한 학습 방법을 말합니다. 일반적으로는 온라인 학습과 오프라인 학습이 합쳐진 학습 방법을 말합니다. 2010년 미국 교육부에서 실시한 블랜디드 러닝의 메타 분석에 따르면 100% 오프라인 또는 온라인 강의 보다 효과적인 것으로 나타났습니다.

# 블렌디드 러닝 신청

강의 신청서 (opens new window)를 남겨주시면 확인 후, 메일 또는 통화를 통해 강의 일정 및 장소, 진행 방향에 대해 답변 드립니다. 😄

블렌디드 러닝 = 소규모 집중 케어 강의

온라인 강의 뿐만 아니라, 오프라인 강의 진행도 필요하기에 블렌디드 러닝은 시간, 공간에 대한 제약으로 7명 까지만 신청을 받습니다.
