# Firebase 웹 앱 등록

Firebase Console (opens new window)에서 웹 앱에 Firebase 등록 과정을 수행한 후, Firebase를 구성하는 코드를 추가합니다. (영상 01:02 까지만 참고)

# Firebase SDK 구성

firebase (opens new window) 패키지를 프로젝트에 설치합니다.

npm i firebase
1

Firebase 앱 초기화 및 인증, 데이터베이스 객체를 생성하여 내보내는 firebase.js 파일을 작성합니다.

src/services/firebase.js

// Firebase 앱 모듈 불러오기
import firebase from "firebase/app"
// 데이터베이스, 인증 모듈 불러오기
import "firebase/firestore"
import "firebase/auth"


// Firebase 구성(Config)
// Firebase Console에서 구성 값을 가져와 이 곳에 붙여 넣습니다.
const config = {
  apiKey: "AIzaSyA0zozq0tAcSqppM0MDUfGUk5aXXXXXXXXX",
  authDomain: "learning-guide-firebase.firebaseapp.com",
  databaseURL: "https://learning-guide-firebase.firebaseio.com",
  projectId: "learning-guide-firebase",
  storageBucket: "learning-guide-firebase.appspot.com",
  messagingSenderId: "474750000000",
  appId: "1:000000000000:web:de4c020000000fdde83x3k"
}

// Firebase 앱 초기화
firebase.initializeApp(config)

// Firebase 인증, 데이터베이스 객체 생성 (내보내기)
export const auth = firebase.auth()
export const db = firebase.firestore()

// Firebase 모듈 기본 내보내기
export default firebase

[오류 발생] Firebase: No Firebase App '[DEFAULT]' has been created

Firebase 앱 초기화 호출 오류가 발생하면 아래처럼 초기화 코드를 수정한 후 실행해 문제를 해결합니다.

// Firebase 앱이 실행 중인 상태 확인
!firebase.apps.length ? 
  // 실행 중인 앱이 없을 경우, Firebase 앱 초기화
  firebase.initializeApp(config) : 
  // 실행 중인 앱이 있을 경우, Firebase 앱 구동
  firebase.app()
