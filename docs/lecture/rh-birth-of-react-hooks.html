# React 훅의 등장

일반적으로 사용자와 인터랙션 하는 애플리케이션은 어딘가에서 상태(state)를 관리해야 합니다. React는 이러한 상태 관리를 위해 훅(hooks) 이라는 특수한 함수를 제공합니다. React에 내장된 훅은 다음과 같습니다.

# 기본 훅

  • React.useState() (opens new window)
  • React.useEffect() (opens new window)
  • React.useContext() (opens new window)

# 추가 훅

  • React.useReducer() (opens new window)
  • React.useCallback() (opens new window)
  • React.useMemo() (opens new window)
  • React.useRef() (opens new window)
  • React.useImperativeHandle() (opens new window)
  • React.useLayoutEffect() (opens new window)
  • React.useDebugValue() (opens new window)

각각의 React 훅은 상태(state)를 저장하거나, 사이드 이펙트(effect)를 수행하기 위한 React 함수 컴포넌트에서 호출 할 수 있는 특수한 함수를 말합니다. 각 훅마다 고유한 사용법(API)이 다릅니다.

예를 들어 상태 및 상태 업데이트 함수를 반환하는 React.useState(), React.useReducer() 그리고 특정 값을 반환하는 React.useRef(), React.useContext() 또 React.useEffect() 처럼 아무런 값도 반환하지 않는 훅이 있습니다.


React 훅 등장(v16.8) 이전에는 용도에 따라 함수 컴포넌트와 클래스 컴포넌트로 분류되어 사용되었습니다.
