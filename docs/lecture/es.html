# ECMAScript

React는 다른 프레임워크와 달리 별도의 템플릿 문법에 의존하지 않고 JavaScript를 적극적으로 활용합니다. 물론 사용자 편의를 위해 JSX 문법을 지원하며 JSX 코드는 컴파일 과정에서 JavaScript 코드로 변환됩니다. 뿐만 아니라 React Hook (opens new window)의 등장으로 컴포넌트는 이전 보다 이해하고 작성하고 사용하기 쉬워졌습니다.

그런 이유로 React를 애플리케이션 개발에서 효율적으로 활용하기 위해서는 모던 JavaScript(ES6+)의 새로운 기능을 배우고 적극적으로 활용해야 합니다. 여기에서 소개하는 새로운 기능들은 React를 최대한 효과적으로 활용하기 위해 반드시 알아두어야 할 JavaScript의 기능입니다.
